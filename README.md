- 出退勤や入退室を打刻して出席率をみたりできるスプレッドシートバインド型のGASウェブアプリです。
- セキュリティポリシーで同一ドメインユーザーのメールアドレス取得が許可されているGoogle Workspace環境での運用を前提としています。
- 各機能の利用可否は組織の制約に依存します。
